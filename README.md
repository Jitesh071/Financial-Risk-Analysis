# 💰 Financial Risk Analysis

A Python-based data analytics project that analyzes banking transaction data to identify financial risks, customer behavior, and transaction patterns. The project performs data cleaning, exploratory data analysis (EDA), customer profiling, anomaly detection, hypothesis testing, and data visualization to generate actionable business insights.

---

## 📖 Project Overview

Financial institutions need to monitor customer transactions to detect unusual behavior, identify inactive accounts, and assess financial risk. This project uses Python to analyze transaction data and provide insights that can support better financial decision-making.

---

## ✨ Features

* Cleaned and formatted raw banking transaction data
* Performed descriptive transaction analysis
* Generated monthly and yearly transaction summaries
* Compared credit and debit transaction trends
* Identified top and bottom performing accounts
* Detected dormant and inactive customer accounts
* Built customer activity profiles (High, Medium, Low)
* Detected unusual and high-risk transactions
* Performed statistical hypothesis testing
* Created insightful visualizations using Matplotlib

## 📊 Analysis Performed

### 1. Data Cleaning

* Checked missing values
* Converted transaction dates
* Validated data types
* Generated descriptive statistics

### 2. Transaction Analysis

* Monthly transaction summaries
* Yearly transaction summaries
* Credit vs Debit comparison
* Net transaction volume

### 3. Customer Profiling

* High Activity (>5 transactions)
* Medium Activity (3–5 transactions)
* Low Activity (<3 transactions)

### 4. Financial Risk Detection

* Dormant account identification
* Inactive account detection
* High withdrawal detection
* Unusual transaction analysis

### 5. Data Visualization

* Credit vs Debit trends
* Transaction summaries
* Customer activity insights

### 6. Hypothesis Testing

* Compared average account balances between customer activity groups using statistical testing.

---

## 📈 Key Findings

* Debit transactions exceeded credit transactions during multiple periods.
* Several dormant and inactive accounts were identified.
* High-value withdrawal patterns indicated potential financial risks.
* Customers were successfully segmented into activity groups.
* Hypothesis testing showed no statistically significant difference in average balances between High and Low activity customers.
